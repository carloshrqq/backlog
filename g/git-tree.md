# LISTAR ARVORE DE ARQUIVOS E GERAR CODEBLOCK EXIBÍVEL
Acesse a pasta raiz do projeto e abra um novo terminal bash

```bash
cmd //c tree //f
```

Copie e cole o código direto do terminar bash para dentro de um codeblock do tipo text.

Por exemplo:
```text
├───A
├───B
│   │   backup.md
│   │   brave-bookmarks.md
│   │
│   └───blueprint
│           blueprint-all-black.jpg
│           blueprint-all-blue.jpg
│           blueprint-desc-ex.jpg
│           blueprint-with-table.jpg
│
├───D
│   │   dailynote-map.md
│   │   data-filename-suggestion.md
│   │   dataview.md
│   │   diagram.md
│   │   diary-deepcon-sentinel.md
│   │   discord-badges.md
│   │   discord-helpers.md

```

Se houver um "." (ponto final) no nome de algum diretório significa que o mesmo é oculto na pasta raiz. Pode ser demonstrado normalmente.
