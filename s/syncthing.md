# Configuração Syncthing

Obsidian + Sincronismo = 100% gratuito! (via Syncthing)
(Tutorial de como funciona, instalação e configuração)
https://www.youtube.com/watch?v=DIzJg3ehYCM

Download Syncthing (pc)
(Necessário para sincronizar arquivos win/lnx/etc)
https://syncthing.net/

Download App Syncthing (mobile)(android ver)
(Necessário para sincronizar dispositivos como celular, tablet, etc)
https://play.google.com/store/search?q=Syncthing&c=apps&hl=pt_BR

Download Material Files - Gerenciador de arquivos (mobile)(android ver)
(Necessário para gerenciar arquivos em dispositivos como celular, tablet, etc sem propagandas, leve)
https://play.google.com/store/apps/details?id=me.zhanghai.android.files&hl=pt_BR

# Inicialização Syncthing

Acessar o endereço no pc
O início é automático uma vez configurado
https://127.0.0.1:8384/

Para iniciar no dispositivo móvel (android) basta abrir o aplicativo. Contudo, por padrão, o aplicativo fica em segundo plano.

_Os diretórios .stfolder() e .stfolder(1) não podem ser excluídos. Caso contrário a configuração deverá ser refeita e a sincronização não acontece._
