> ### ★ Tutorial interface VSCode minimalista
> Segue o link do meu tutorial com referências.
> A primeira parte do tutorial trata de um detalhe de extensão bem fácil de resolver.

Meu [Tutorial interface VSCode minimalista](https://gist.github.com/carloshenriquefnt/8729842e65dcda6974392ddbde0e3389) adaptado

> ### ⚠ Versão 1.93 do VSCode
> Para conseguir concluir as instruções deste tutorial de mudança de interface é necessário instalar a [versão 1.93 do VSCode](https://update.code.visualstudio.com/1.93.1/win32-x64-user/stable).

> ### ☀ settings.json
> Para acessar o arquivo de configuração do VSCode pressione `CTRL`+`SHIFT`+`P` e digite settings.json e clique na primeira opção "*Open user settings (json)*"
>  
> Para acessar as extensões do VSCode pressione `CTRL`+`SHIFT`+`P` e digite `install extensions`e clique na opção "*Extensions: Install Extensions*" e na janela que se abre procure as extensões que deseja instalar (lista completa das extensões no [tutorial](https://gist.github.com/carloshenriquefnt/8729842e65dcda6974392ddbde0e3389)).

> ### ≪/≫ Sem programação
> Não é necessário ter conhecimentos prévios em programação para completar o tutorial.

> ### ✎ Customização
> As extensões tentam abrangir de modo geral certas utilidades em determinados cenários de programação. Fique à vontade para remover ou adicionar extensões ou alterar o arquivo de configuração.
