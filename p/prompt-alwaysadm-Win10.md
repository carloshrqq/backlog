# prompt-always-admWin10

Como abrir prompt de comando do Windows sempre como administrador (inclui flowlauncher)

C:\Users\USER\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\System Tools
Clique direito no atalho "Prompt de Comando" -> Propriedades -> Avançados -> Executar como Administrador.

"Ok" e "aplicar"
