Ativação de plugins de cabeçalho (frontmatter, properties view, properties in document)
https://www.youtube.com/watch?v=LYTbqyxx8So&list=PL-dDA0NKbGQQwqx-Rl9AQSrCeb3ekr4fX&index=3

Config > Core plugins > properties view > ON
Config > Editor > Properties in Document > Visible
Digite três vezes o sinal de - (menos) para abrir as propriedades do cabeçalho
Config > Dataview
Enable Javascript Queries
Enable Javascript inline Queries

DataView
Config > Community Plugins > Browse > DataView
Config > Community Plugins > Installed Plugins > Search plugin name and turn ON

Usar tres crases (´) digitar dataview e dentro especificar o corpo do filtro. Acesse a documentação do plugin para maiores informações.

Templates automáticos
Criar a pasta para os templates. Observação! A pasta deve conter apenas templates.
