## Como instalar

`Configurações` > `Community Plugins` > `Browse` > digite o nome do plugin `calendar` > clique sobre o card do plugin > na nova janela que se abre clique no botão `install`

Ainda na janela `community plugins` __ative__ o plugin na lista `Installed plugins`

Em alguns casos pode ser necessário reinicar o obsidian.

## Configurações específicas dos plugins

### Core plugins

#### dailynote

`Configurações > Daily notes > New File Location` (especifique o local do arquivo) > `Open daily note on startup` (ativar)`

Importante: Se houver pasta no diretório final do local do arquivo a mesma deve ser especificada no endereço e deve existir no caminho (o plugin não cria pastas)

### Community plugins

#### calendar

`Start week on Sunday` > ON
#### dataview

`Enable Javascript Queries` > ON

#### git

`auto commit-and-sync interval (minutes)` > Pode configurar como 7 minutos

`Auto commit-and-sync after stopping flie edits` > ON

#### Templater

`Syntax highlighting` > ON

`Automatic jump to cursor` > ON

`Enable folder templates` > ON
